El proyecto estará integrado por 3 entregables

La definición del primer entregable es:

 

Desarrollar en JAVA 11 un proyecto usando Maven.

 

Presentar un menú al usuario con las siguientes opciones:

Arboles binarios
XX
YY
 

Entregable 1

En caso de seleccionar la opción 1 (Arboles binarios) presentar los siguientes submenús
Cargar archivo a BD
Generar Árbol
Imprimir Árbol
Limpiar base de datos
 

En caso de seleccionar la opción a, debe solicitar al usuario que indique donde está ubicado el archivo de texto a cargar en la base de datos.

 

El formato del txt debe ser: 1,2,3,4,89,789, es decir debe aceptar únicamente números y cada número está separado por una coma (,) si el archivo no tiene el formato indicado debe notificar al usuario.

 

Al finalizar la validación del documento deben insertarlo en una tabla (La base de datos depende del su número de grupo) con tres columnas una que sea un id de la tabla que sea autogenerado por la base de datos, el segundo campo se debe de llamar dato donde depositaran cada uno de los números leídos y por último un campo de estado donde se indique si el valor es 1 que el registro esta activo si es 0 que el registro este inactivo.

 

En caso de seleccionar la opción B, debe leer la tabla populada en la opción A y generar el árbol binario.

 

En caso de seleccionar la opción C, debe mostrar el árbol gráficamente.

 

En caso de seleccionar la opción D, debe eliminar de manera lógica los registros actuales de la base de datos.

 

 
*Nota: En cada opción deben hacer validaciones acordes a los solicitado, por ejemplo, en la opción B, validar que la base de datos tenga datos para generar el árbol, en la opción C deben validar que el árbol esté creado.

 

Base de datos: MySQL
